# YouTube Thumbnail Generator

YouTubeサムネイル・バナー生成ツール

## 概要

シンガーソングライター「彩瀬こよみ」のYouTube活動用サムネイル・バナー生成ツールです。

## 機能

- テンプレート選択（新曲発表/カバー曲/歌ってみた/MV公開/LIVE告知）
- キャラクター画像の配置
- テキスト編集
- レイアウト調整
- 背景設定（単色/画像/AI生成）
- PNG/JPEG出力（1280x720px）

## 必要環境

- Python 3.10+
- CustomTkinter
- Pillow
- PyYAML
- google-genai

## 起動方法

```bash
# macOS/Linux
./run_app.command

# または
python3 app/main.py
```
